## METHODOLOGY CONCERNS

- **The matched-response-model test in 4.3.1 uses 6 judges but excludes GPT-5.4 due to 400 errors.** While reasonable, the exclusion should be documented more transparently: Was GPT-5.4 tested on other conditions? Was the error rate consistent? This affects cross-model comparability.

- **The stateful-agent test (4.3.1) uses a single subject (Hamerton) and a single run.** While justified by cost, this limits generalizability. The paper should explicitly state that no replication or variance analysis was performed on the Letta agent’s memory block content.

- **The “matched response model” test does not control for context length.** Letta’s block is ~5.6K tokens, Base Layer’s spec ~8.5K. The paper notes this, but does not attempt normalization (e.g., truncating the spec, expanding the block). This leaves open whether performance differences are due to content or capacity.
