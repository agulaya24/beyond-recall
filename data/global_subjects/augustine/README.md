# `data/global_subjects/augustine/` — Input data for Augustine of Hippo

**What's in this folder:** The source facts, question battery, and behavioral-specification layers for Augustine of Hippo (North African / Roman, 354-430, C5 baseline 2.79). Source text: *Confessions*.

Higher-baseline subject. The model already knows Augustine well, so the spec effect is expected to be modest.

## Contents

Standard per-subject schema. See `../README.md` for the full file list and meanings. In brief: `facts.json`, `battery.json`, `spec.md`, `spec_production.md`, `anchors_v4.md`, `core_v4.md`, `predictions_v4.md`, `brief_v5.md`, plus subject-level `results.json` and `judgments.json`.

## Where these files come from / go to

Inputs: Augustine's *Confessions* (public domain, Project Gutenberg). Outputs feed `results/global_augustine/`.
