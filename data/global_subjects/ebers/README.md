**What's in this folder:** The source facts, question battery, and behavioral-specification layers for Georg Ebers (German, 1837-1898, C5 baseline 1.04). Source text: *Story of My Life*.

One of the lowest-baseline subjects. Used in the Letta stateful-agent test (paper §4.7).
